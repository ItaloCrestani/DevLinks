<h1 align="center"> 🔗 DevLinks </h1>

Um agregador de links, desenvolvido para centralizar e compartilhar seus principais links em um só lugar.

## 🚀 Tecnologias utilizadas

- React + TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Firebase

## 📌 Sobre o projeto

O DevLinks é uma aplicação web que permite exibir uma lista personalizada de links em uma página pública, enquanto oferece uma área administrativa protegida para gerenciamento desses links.

A proposta é ser uma alternativa simples e eficiente para páginas de bio com links (como Linktree), com foco em aprendizado e prática de tecnologias modernas.

## 📂 Funcionalidades

- Exibição de links na página principal
- Sistema de login com Firebase
- Área administrativa protegida
- Adição de novos links
- Remoção de links
